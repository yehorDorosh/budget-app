# Build

Before build don't forget set server domain `EXPO_PUBLIC_API_URL` in `eas.json`

## Local

`eas build -p android --profile preview --local`

## EAS

`eas build -p android --profile preview`

# DEV

## Run

`npm start`

## TypeScript check all

`npm run ts:check`

## React Dev Tool

`npm run react-devtools`
